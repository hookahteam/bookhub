# BookHub  

Добро пожаловать на страницу BookHub. Наш сайт - это каталог популярных книг, взятых из открытых источников.  
Все книги доступны для чтения, скачивания, цитирования и копирования бесплатно, без проблем с авторскими правами.

**Функционал**:  
На данный момент на сайте доступна:  
Регистрация/Логин

**Стек:**  
Бэкенд:  
Серверное приложение сайта написано по принципам REST  
Использовался язык С++ и фреймворк [Crow](https://github.com/crowcpp/crow) для удобной обработки HTTP запросов  
Библиотека [pqxx](https://github.com/jtv/libpqxx) (обертка для PostgreSQL) для создания баз данных  
[Bcrypt](https://github.com/trusch/libbcrypt) и OpenSSL для хеша паролей и генерации/валидации JWT токенов

**Сборка:**  
Linux/MacOS:  
Установите пакеты git и cmake  

```bash
git clone https://github.com/hookahteam/bookhub/
./scripts/setup.sh
```

Для запуска используйте команду в корневой директории:  
```bash
make release
```
